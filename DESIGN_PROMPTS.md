Companion to `PRODUCTS_ROADMAP.md`. For each ⏳ Planned product, this file lists:

1. **Printful blank** — what to search for in Printful's catalog
2. **Print placement / size** — where the design goes and how big

4. **AI prompt** — paste into Midjourney / DALL·E / Stable Diffusion
5. **Notes** — gotchas, alternates

## Shared brand visual system

Use this across every design so the store feels like one brand, not 22 random tees.

- **Palette:** hot pink `#ff007a`, cream `#f8eaea`, deep black `#0a0a0a`, accent "wine drunk" `#7a0033`, pearl/silver highlights
- **Mood:** Y2K nostalgia × modern feminine × slightly cynical. Glossy. Sticker-cute meets villain era.
- **Typography (add in Canva/Figma after generation):** chunky display sans (Druk, Cooper Black, GT Maru) for headlines; condensed/italic accents (Migra, Editorial New) for detail
- **Motifs:** broken hearts, butterflies, pearls, dagger, snake, lipstick smudges, melting wax, flame, halo, sparkles, glossy gradient highlights
- **Composition rule:** centered "sticker-style" lockup, ~80% canvas fill, lots of negative space → reads clean on shirts, mugs, totes alike

## How to use AI prompts in this file

**Generate illustrations in AI; add real text in Canva/Figma after.** Midjourney and DALL·E both still mangle typography on small text. Generate the visual elements with the prompt, then layer the actual headline text using your chosen font.

- **Midjourney v6/v7:** append `--ar 1:1 --v 7 --style raw --no mockup, frame, watermark`
- **DALL·E (ChatGPT):** prepend `"Create a print-ready design illustration with transparent background. "` to each prompt
- **Stable Diffusion / Flux:** add `transparent png, vector style, flat illustration, 4k, sticker design`
- **Always:** export 300 DPI PNG with transparent background

## Workflow: from this file to a live product

For each row above:

1. Open the **AI prompt** in your tool of choice → generate 3-5 variations → pick best
2. Open in **Canva or Figma** → layer the actual typography over the illustration using a real font
3. Export **300 DPI PNG, transparent background**, save to `designs/` as `p{N}-{slug}.png`
4. Upload to **Printful** → pick the **blank** named above → set placement & size as listed
5. Publish → copy each variant's `sync_variant_id`
6. Come back to me with the IDs and I'll wire them into `lib/printful-mapping.js`
7. Run a test order with the Stripe test card (`4242 4242 4242 4242`) → confirm the webhook creates a Printful draft order
